# Menü manuell erstellen

Das Test-Menü konnte nicht automatisch erstellt werden. Bitte erstellen Sie es manuell:

## Schritt-für-Schritt:

1. **Gehen Sie zu:** Online Store → Navigation
2. **Klicken Sie auf:** "Menü hinzufügen"
3. **Geben Sie ein:**
   - **Name:** `Test Menu`
   - **Handle:** `test-menu` (WICHTIG - muss genau so sein!)
4. **Fügen Sie Menüpunkte hinzu:**
   - Klicken Sie auf "Menüpunkt hinzufügen"
   - **Startseite:**
     - Name: `Startseite`
     - Link: `/`
   - **Test Kollektion:**
     - Name: `Test Kollektion`
     - Link: `/collections/test-kollektion`
   - **Alle Produkte:**
     - Name: `Alle Produkte`
     - Link: `/collections/all`
5. **Speichern**

## Menü im Theme zuweisen:

1. Gehen Sie zu: **Online Store → Themes → Anpassen**
2. Klicken Sie auf **Header** Section
3. Wählen Sie unter **Menu** das **"Test Menu"** aus
4. **Speichern**

## Fertig! ✅

Jetzt sollten alle Test-Daten erstellt und konfiguriert sein!
